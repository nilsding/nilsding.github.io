View On GitHub

    Willkommen Sie auf GitHub.

    Diese automatische Seiten-Generator ist die einfachste Möglichkeit zum Erstellen von schöner Seiten für alle Ihre Projekte. Autor, Inhalt Ihrer Seite hier mit GitHub gewürzte Markdown, wählen Sie eine Vorlage von einem Designer gefertigt und veröffentlichen. Nachdem Ihre Seite generiert wurde, können Sie den neuen Zweig Auschecken:

    $ cd Your_repo_root/repo_name
$ Git Abruf Herkunft
$ Git gh-Kaufabwicklungsseiten

    Wenn Sie die GitHub für Mac verwenden, synchronisieren Sie einfach Ihr Depot und Sie sehen die neue Niederlassung.

    Designer Vorlagen

    Wir haben einige schönen Vorlagen für die Verwendung in Handarbeit gemacht. Gehen Sie voran und weiterhin Layouts, um durch sie zu durchsuchen. Sie können leicht zurück gehen, um Ihre Seite vor der Veröffentlichung zu bearbeiten. Nach der Veröffentlichung Ihrer Seite, können Sie überarbeiten den Seiten-Generator und wechseln Sie zu einem anderen Thema. Ihre Seiteninhalte werden wenn es Markdown-Format blieb beibehalten.

    Eher fahren Stick?

    Wenn Sie nicht den automatischen Generator verwenden möchten, drücken Sie einen Zweig mit dem Namen "gh-Seiten" in Ihr Repository eine Seite manuell zu erstellen. Zusätzlich zur Unterstützung von regulären HTML-Inhalt, support-Seiten von GitHub Jekyll, eine Simple, Blog bewusst statische Website Generator geschrieben von unseren eigenen Tom Preston-Werner. Jekyll erleichtert die standortweite Kopf- und Fußzeilen zu erstellen, ohne dass sie auf jeder Seite kopiert. Es bietet auch intelligente Blog-Unterstützung und andere erweiterte Template-Funktionen.

    Autoren und Mitwirkende

    Sie können @mention ein GitHub Username, einen Link zu ihrem Profil zu generieren. Die daraus resultierende "" Element des Teilnehmers GitHub Profil verknüpfen wird. Zum Beispiel: GitHub gegründet 2007, Chris Wanstrath (@defunkt), PJ Hyett (@pjhyett) und Tom Preston-Werner (@mojombo).

    Unterstützen oder kontaktieren

    Haben Sie Probleme mit Seiten? Schauen Sie sich die Dokumentation unter http://help.github.com/pages oder Kontakt support@github.com und wir helfen Ihnen es zu klären.
